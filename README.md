# 🏭 Supply Chain Stress Forecasting Platform

> End-to-end ML platform that forecasts supply chain stress by modeling how disruptions propagate through supplier networks, combining graph analytics, NLP, macro indicators, and a RAG-powered assistant.

---

## 🎯 Problem
Supply chain disruptions don't happen in isolation — stress at a tier-1 supplier like TSMC cascades downstream to NVIDIA, Apple, and automotive manufacturers days before it shows up in earnings reports. Most tools treat companies as independent signals. This platform models the actual network structure.

---

## 🚀 What It Does
- Monitors **25 public companies** across 6 sectors in real time
- Models supplier relationships as a **directed graph** and propagates stress signals downstream
- Achieves **96.78% AUC** with XGBoost using stock volatility, macro indicators, and NLP sentiment
- Forecasts aggregate stress **30 days forward** using Ridge regression
- **RAG Q&A assistant** powered by Llama 3.3 70B answers natural language questions grounded in live news and model outputs

## ⚙️ Setup

### 1. Clone the repo
```bash
git clone https://github.com/yourusername/supply-chain-stress-forecasting.git
cd supply-chain-stress-forecasting
```

### 2. Create virtual environment (Python 3.12)
```bash
# Windows
py -3.12 -m venv venv
venv\Scripts\Activate.ps1

# Mac/Linux
python3.12 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies
```bash
# Install PyTorch first (Windows)
pip install torch --index-url https://download.pytorch.org/whl/cpu

# Install everything else
pip install -r requirements.txt
python -m spacy download en_core_web_sm
pip install faiss-cpu langchain-huggingface
```

### 4. Set up API keys
Create a `.env` file in the project root:
```
GROQ_API_KEY=your_groq_key        # console.groq.com (free)
NEWS_API_KEY=your_newsapi_key     # newsapi.org (free)
FRED_API_KEY=your_fred_key        # fred.stlouisfed.org (free)
```

### 5. Run the pipeline
```bash
python ingest_pipeline.py
```

### 6. Launch the dashboard
```bash
streamlit run src/dashboard/app.py
```
